# ドキュメント

このディレクトリには、プロジェクトのスクリーンショットや追加ドキュメントを配置します。

## スクリーンショット

以下のスクリーンショットを配置してください：

- `screenshot-input.png`: 入力画面のスクリーンショット
- `screenshot-result.png`: 結果画面（スコア/引用/改善案）のスクリーンショット

### スクリーンショットの取得方法

1. Streamlitアプリを起動
2. 入力画面を表示してスクリーンショットを取得 → `screenshot-input.png` として保存
3. 分析を実行して結果画面を表示 → `screenshot-result.png` として保存
